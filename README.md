# DevOps Assignment 1 - Version Control with Git and GitHub

## Project Description
This project demonstrates the fundamentals of version control using Git and GitHub. It covers essential concepts including:

- Repository initialization
- Committing changes
- Branch creation and management
- Remote repository integration
- Pull requests and merging

## Purpose
This assignment is part of Module 1 coursework focusing on Git and GitHub workflows, designed to build practical skills in collaborative development practices.
